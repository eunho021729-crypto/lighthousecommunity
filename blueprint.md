# Blueprint: 등대마을공동체 웹사이트

## 1. 프로젝트 개요

**등대마을공동체** 웹사이트는 마을 주민들에게 다양한 커뮤니티 공간을 소개하고, 후원자와 수혜자를 연결하는 따뜻한 복지 플랫폼을 제공하는 것을 목표로 합니다. 오프라인 공간의 활성화와 온라인을 통한 나눔 문화 확산을 동시에 추구하는 허브 역할을 합니다.

---

## 2. 디자인 및 기능 명세

### 가. 디자인 컨셉 (v2)

-   **Modern & Simple:** 깔끔하고 현대적인 'SUIT' 폰트를 메인으로 사용하여 가독성을 높이고 세련된 느낌을 줍니다.
-   **Iconography:** 로고에 등대(💡) 아이콘을 사용하여 정체성을 상징적으로 표현합니다.
-   **Color Palette (Beige/Green):**
    -   `--primary-color: #5d7055` (차분한 연두색)
    -   `--accent-color: #e9e5d7` (부드러운 베이지색)
    -   `--bg-color: #fbfaf6` (아주 밝은 배경 베이지)
    -   `--text-color: #3d3d3d` (짙은 회색)
-   **Layout:** CSS Grid(2x2)와 Flexbox를 활용한 반응형 웹 디자인.
-   **Components:** 카드형 UI와 명확한 CTA 버튼으로 직관적인 사용자 경험을 제공합니다.

### 나. 핵심 기능

#### 1) 공통 기능
-   **로그인 기능:**
    -   헤더에 '로그인' 버튼을 배치합니다.
    -   사용자는 `login.html` 페이지를 통해 시스템에 접근할 수 있습니다. (현재는 UI만 구현)

#### 2) 마을 공간 소개 (`village-space`)
-   **네이버 스마트플레이스 연동:** 섹션 제목 옆에 작은 버튼을 추가하여 네이버 지도/플레이스 페이지로 연결합니다.
-   **카페앤디저트:** 메뉴를 소개하는 `menu.html` 페이지로 연결됩니다.
-   **마을서재:** 책과 보드게임을 즐길 수 있으며, 보유 리스트를 `list.html` 페이지에서 확인할 수 있습니다.
-   **커뮤니티 공간:** 대관 문의를 위한 `rental.html` 폼 페이지로 연결됩니다.
-   **마을 정보 사무실:** 마을의 최신 소식을 전하는 `info.html` 페이지로 연결됩니다.

#### 3) 참여와 나눔 플랫폼 (`participation-and-sharing`)
-   **후원하기:** 후원 정보를 입력하는 `donation.html` 폼 페이지로 연결됩니다.
-   **후원 신청하기:** 지원 요청 양식을 작성하는 `support_request.html` 폼 페이지로 연결됩니다.
-   **공동체 후기:** 프로젝트 펀딩, 마을 사업, 공간 이용 및 굿즈 구매 등 공동체 활동 전반에 대한 후기를 남기고 공유하는 `reviews.html` 게시판 페이지로 연결됩니다.

## 3. 현재 개발 계획 (Phase 4: 디자인 리프레시)

1.  **`blueprint.md` 업데이트 (완료):** 폰트를 'SUIT'로 변경하고, 색상 테마를 베이지/연두 계열로 수정했습니다.
2.  **`style.css` 수정:** 새로운 색상 팔레트를 CSS 변수에 적용하고, 기본 폰트 설정을 변경합니다.
3.  **전체 HTML 파일 수정:** '나눔손글씨 펜' 폰트 링크를 제거하고, 'SUIT' 폰트가 올바르게 적용되도록 확인합니다.
4.  **`preview.html` 최종 확인:** 모든 디자인 변경 사항이 반영된 `preview.html` 파일을 업데이트하여 최종 결과물을 확인합니다.
